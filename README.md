# 连连看小游戏

一个使用HTML、CSS和JavaScript实现的经典连连看小游戏。


## 文件结构

```
lianliankan/
├── index.html      # HTML 结构
├── style.css       # 样式表
├── game.js         # 游戏逻辑
└── README.md       # 说明文档
```

## 快速开始

1. 先下载安装一个vs code，然后在自己的电脑中右击解压缩的项目文件，选择用vs code打开
2. 在浏览器中打开 `index.html` 文件
2. 点击棋盘上的相同图片块
3. 消除所有的方块即可通关



## 游戏按钮说明

- **新游戏**：开始一局新游戏
- **撤销**：撤销上一次消除操作
- **提示**：显示一对可以消除的方块
- **设置**：调整游戏难度和图片主题


右下角有一个音乐图标，点击可以开启或关闭背景音乐，双击可以调整音量大小.
连续消除10对触发安全车，倒计时会暂停5秒
连续三次错误会触发罚时，时间减少3秒
